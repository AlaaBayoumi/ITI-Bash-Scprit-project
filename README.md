# ITI-Bash-Scprit-project
A simple Bash-based user management system that simulates basic system administration tasks through a menu-driven interface.

📌 Overview

This project provides a command-line interface (CLI) for managing users and groups. It allows administrators to perform essential operations مثل إضافة المستخدمين، تعديلهم، حذفهم، وإدارة المجموعات بسهولة.

⚙️ Features
➕ Add User
✏️ Modify User
❌ Delete User
📋 List Users
👥 Add Group
✏️ Modify Group
❌ Delete Group
📋 List Groups
🔒 Disable User (Lock account)
🔓 Enable User (Unlock account)
🔑 Change User Password
